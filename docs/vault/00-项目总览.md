# 高纳AI 项目总览

## 定位

面向工业 AOI 的少样本缺陷检测数据闭环平台。支持三种路线：
- 人工标注 + 监督训练
- AI 预标注 + 人工审核
- OK 样本异常检测

## 技术栈

| 层 | 技术 |
|---|---|
| 前端 | React 18 + TypeScript + Vite + Tailwind CSS + Fabric.js + Recharts |
| 后端 | Python FastAPI + SQLModel |
| 数据库 | SQLite (MVP) / PostgreSQL (生产) |
| 认证 | JWT |
| AI | OpenCV + sklearn (CPU) / YOLO + PyTorch (可选) |

## 配置项

`.env`:

| 变量 | 说明 | 默认值 |
|---|---|---|
| HTX_APP_NAME | 软件名称 | 高纳AI |
| HTX_COMPANY_NAME | 公司名称 | 高纳科技 |
| HTX_PLATFORM_NAME | 平台名称/副标题 | 工业视觉平台 |
| HTX_THEME | 主题 (light/dark/industrial) | light |
| HTX_LAYOUT | 布局 (sidebar/topbar/focus) | sidebar |
| HTX_HOST_BIND | 宿主机监听地址 | 0.0.0.0 |
| HTX_FRONTEND_PORT | 前端宿主机端口 | 80 |
| HTX_BACKEND_PORT | 后端宿主机端口 | 8000 |
| HTX_SAM_MODEL_SIZE | 默认 SAM2 模型 | tiny |
| HTX_SAM_MODELS_DIR | 容器内 SAM2 模型目录 | ./models |

## 快速启动

```bash
# 后端
cd backend && python seed.py && uvicorn app.main:app --reload --port 8000

# 前端
cd frontend && npm run dev
```

也可以在启动命令里直接覆盖分发参数：

```bash
HTX_APP_NAME=某某AI HTX_COMPANY_NAME=某某科技 HTX_PLATFORM_NAME=质检云平台 HTX_THEME=industrial HTX_LAYOUT=topbar HTX_HOST_BIND=0.0.0.0 HTX_FRONTEND_PORT=8080 HTX_BACKEND_PORT=18000 bash scripts/up.sh
```
