# 主题配置

三套主题，通过 `.env` 的 `THEME` 变量切换。

## light

默认。白底灰字，zinc-950 深色侧边栏。

```env
THEME=light
```

## dark

黑底白字，全暗模式。

```env
THEME=dark
```

## industrial

AOI 风格。深绿底 + teal 强调色 + 等宽字体。

```env
THEME=industrial
```

## 切换方式

1. 改 `.env` 里的 `THEME=dark` 或 `THEME=industrial`
2. `docker compose up -d --build`
3. 刷新浏览器

## 自定义主题

编辑 `frontend/src/index.css`，添加新的 `[data-theme="xxx"]` 块。CSS 变量列表：

| 变量 | 说明 |
|---|---|
| --bg-main | 页面背景 |
| --bg-card | 卡片背景 |
| --bg-sidebar | 侧边栏背景 |
| --text-primary | 主文字 |
| --text-secondary | 次文字 |
| --border | 边框颜色 |
| --accent | 强调色(按钮、链接) |
| --header-bg | 顶栏背景 |
| --input-bg | 输入框背景 |
