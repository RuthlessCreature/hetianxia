# 架构设计

## 系统架构图

```
浏览器(React) → Nginx:80 → /api → FastAPI:8000 → SQLite
                            /static/
```

## 数据流

### 标注 → 训练

```
上传图片 → ImageAsset(DB)
         → Annotation(DB) [human | ai_prelabel]
审核队列 → status: candidate → confirmed/rejected
数据集版本 → freeze → DatasetItem(train/val/test 切分)
训练任务 → mock_training / real_training → metrics + ModelRegistry
评估 → EvaluationReport(metrics + failure_cases)
```

### 预标注流程

```
用户点"AI预标注"
  → PrelabelJob(queued)
  → generate_prelabel(image_ids, method='hybrid')
    → YOLOv8-nano (if torch) ← 可选
    → OpenCV Canny/DoG/Harris
  → Annotation(candidate) × N
  → 标注页审核 → confirmed/rejected
```

## 数据模型 (14表)

| 表 | 用途 |
|---|---|
| tenants | 租户/客户 |
| users | 用户(rbac) |
| projects | 项目 |
| image_assets | 图片 |
| annotations | 标注(candidate/confirmed/rejected) |
| dataset_versions | 数据集版本 |
| dataset_items | 数据集条目(train/val/test) |
| prelabel_jobs | AI预标注任务 |
| training_jobs | 训练任务 |
| model_registry | 模型注册表 |
| evaluation_reports | 评估报告 |
| audit_logs | 操作日志 |
| deployments | 部署记录 |
| notifications | 通知 |

## 安全

- JWT 认证(token含 tenant_id)
- RBAC: owner > admin > engineer > labeler > viewer
- 租户隔离(所有查询带 tenant_id)
- License 过期检查
- 用量限制(项目数/图片数/训练次数)
- API 限流(120次/分钟/IP)
- 预标注 candidate ≠ ground truth(必须人工确认)
