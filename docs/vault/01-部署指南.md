# 部署指南

## 服务器要求

- Linux (Alibaba Linux 3 / Ubuntu 20.04+)
- Docker + Docker Compose
- 内存 >= 1GB
- 端口 80 开放

## 完整流程

### 1. 装 Docker

```bash
# Alibaba Linux 3
dnf config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
dnf install -y docker-ce docker-compose-plugin

# 国内镜像加速
cat > /etc/docker/daemon.json << 'EOF'
{"registry-mirrors":["https://docker.1panel.live","https://hub.rat.dev"]}
EOF
systemctl daemon-reload && systemctl enable --now docker
```

### 2. 拉代码

```bash
cd /root
git clone <仓库地址> hetianxia
cd hetianxia
```

### 3. 配置

```bash
cat > .env << EOF
APP_NAME=高纳AI
COMPANY_NAME=高纳科技
THEME=light
DATABASE_URL=sqlite:///./data/hetianxia.db
SECRET_KEY=$(openssl rand -hex 32)
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=1440
UPLOAD_DIR=./data/uploads
THUMBNAIL_DIR=./data/thumbnails
EOF
mkdir -p data/uploads data/thumbnails data/models
```

### 4. 启动

```bash
docker compose up -d --build
docker compose exec backend python seed.py
```

### 5. 验证

```
http://服务器IP
```

## 运维命令

```bash
docker compose ps              # 状态
docker compose logs -f         # 日志
docker compose restart         # 重启
docker compose down            # 停止
docker compose up -d --build   # 重建
```

## 升级

```bash
git pull
docker compose up -d --build
```

## 改名字

编辑 `.env`:
```
APP_NAME=你的产品名
COMPANY_NAME=你的公司名
THEME=dark
```
然后 `docker compose up -d --build`
