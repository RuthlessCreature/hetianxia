# 使用手册

## 角色权限

| 角色 | 权限 |
|---|---|
| Owner | 全部权限 |
| Admin | 管理项目、成员 |
| Engineer | 上传、标注、训练 |
| Labeler | 标注和审核 |
| Viewer | 只读 |

## 三种检测路线

### 路线一：人工标注 + 监督训练

上传图片 → 人工画框/标OK NG → 创建数据集 → 冻结 → 训练 → 评估 → 模型

### 路线二：AI 预标注 + 人工审核

上传图片 → AI 预标注(OpenCV+YOLO) → 审核候选框 → 确认/拒绝 → 进入训练

### 路线三：OK 样本异常检测

上传OK图 → 训练正常分布模型 → 测试 → 异常分数 → 阈值判定

## 标注画布

### 三种模式

| 模式 | 操作 |
|---|---|
| 框 | 拖空白画框 / 点已有框选中 |
| 多边形 | 点空白加顶点 / 点"闭合"完成 |
| 选择 | 选中移动 / Delete删除 / 右侧下拉改标签 |

### 审核

候选标注(橙色虚线) → 点"确认"变绿实线 / 点"拒绝"删除

### 自定义标签

左侧标签列表底部输入框 → 输入新标签名 → 回车或点+。存在 localStorage 中。

## 训练

1. 上传并标注图片
2. 训练页 → 快速创建并冻结数据集
3. 选任务类型 → 启动训练
4. 完成后自动生成模型记录

## 评估

1. 选择模型版本
2. 调节阈值(0-1滑块)
3. 运行评估
4. 查看 Precision / Recall / F1 / 误报漏报 / 失败样本
5. 导出 JSON 或 Markdown 报告

## 模型管理

- 在线测试：模拟推理
- 导出模型
- 部署到推理端点
- 回滚到上一版本
- 多模型对比
